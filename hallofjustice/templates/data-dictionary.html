Data Dictionary

State

the state the data relates to

Sublocation - city, county, or other local area name, unless the data is ‘Statewide’

Category - primary category that the data falls into, per the state research schema (see below)

Subcategory - secondary category that the data falls into, per the state research schema (see below)

Title - title of the dataset, report, or paper

Description - short description of the contents of the dataset, report, or paper

Format - file type that the data is stored in (.pdf, .xls, .csv, .xml, etc…)

URL - hyperlink to the referenced data

Private/Government - whether the data was created by a private, non-profit, or government (public) entity

Group Name - name of the organization, agency, or group that created the data

Location - if a direct link to the data is not available, a description of where data can be found on the linked page

Associated legislation - legislation associated with the data or that mandated the creation of the data, if applicable

Internet availability? - whether the data can be found on the Internet (Yes/No)

Population data? - whether the data includes population information (Yes/No)

Mappable? - whether the data can be mapped, requires location information (Yes/No)

Is this data updated? - whether the data has been updated recently (Yes/No)

Frequency - how often the data is updated (annually, monthly, daily, etc…)

Data range - the time period the data covers (in years), or the year the report was published

Associated grant - federal or state grant that funded the creation of the data, if applicable

Tag - related terms, to be used in search functionality

Access type

the level of data accessibility:

Open

data is accessible free of charge via the Internet and downloadable in a format that allows for the data to be easily searched/sorted

Not machine readable

data is accessible free of charge via the Internet and downloadable in a format that cannot be searched/sorted

Not machine processable

data is not accessible via the Internet or is not delivered in a digital format

Restricted

data is released only to certain groups (i.e. academics, practitioners)

Closed

data is not released or requires payment to be accessed
